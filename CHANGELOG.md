## v2.1.0 (2026-02-19)

- 新增「黑白名单拦截时提醒」开关，被拦截的红包也会发送提醒消息
- 修复配置未正确持久化到 data 目录的问题
- 新增 GitHub Actions 自动打包发布
- 回馈消息发送目标扩展为四种：自己、我的手机、QQ好友、群聊
- 修复插件说明页面项目地址无法点击的问题
- 优化红包抓取逻辑与通知机制
- 移除硬编码窗口ID以提升跨平台兼容性

## v2.2.0 (2026-04-07)

- 新增不抢自己发送的红包（antiMyself）功能
- 新增启用随机发回复延迟开关
- 新增 `%msgTime%`、`%standardTime%` 替换符，支持在回馈消息中显示时间
- 优化自定义回馈消息输入框
- 优化发送回复判断，已抢红包后因未知原因（如断网后恢复）导致间隔过长将不再发送感谢消息
- 修复默认回馈消息模板中多余"的"的问题
- 修复感谢消息延迟开关（useRandomDelayForSend）未独立生效的问题
- 移除硬编码窗口 ID，修复 Linux 平台插件失效的问题
